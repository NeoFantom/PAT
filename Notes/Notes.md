## Python tricks

| 变量  | 类型（或含义） | 变量 | 类型（或含义）  |
| ----- | -------------- | ---- | --------------- |
| `li`  | list           | `di` | dict            |
| `tu`  | tuple          | `s`  | string          |
| `rli` | reversed list  | `rs` | reversed string |
| `t`   | temp           |      |                 |

1. 对列表（或者任何可以按序遍历的东西）进行反向操作  
`rli = li[::-1]`
1. 打印两个整数 a 和 b  
`print(f'{a} {b}')`
